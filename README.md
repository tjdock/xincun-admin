## 一.安装

--save 或者不写参数的话会安装到<br>
"dependencies"<br>
--save-dev 安装到<br>
"devDependencies"<br>

### 1.antd

[ant ui 官网](https://ant.design)

`npm i antd`

### 2.sass 编译器

`npm i node-sass`

### 3.axios

[axios 官网](https://github.com/axios/axios)

`npm i axios`

### 4.路由

`npm i react-router-dom`

### 5.prop-types

`npm i prop-types`

### 6.redux

`npm i redux react-redux redux-saga redux-thunk`

### 7.调试工具

`npm i redux-devtools-extension --save-dev`

### 8.支持 IE11

`npm i @babel/polyfill`
